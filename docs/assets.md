## Спрайты

### Персонажи (Characters)

#### Игрок

**Размер:** 48x64 px (ширина x высота)

**Анимации:**
- `idle-down` — простой (4 кадра)
- `idle-up` — простой спиной (4 кадра)
- `idle-left` — простой влево (4 кадра)
- `idle-right` — простой вправо (4 кадра)
- `walk-down` — ходьба вниз (6-8 кадров)
- `walk-up` — ходьба вверх (6-8 кадров)
- `walk-left` — ходьба влево (6-8 кадров)
- `walk-right` — ходьба вправо (6-8 кадров)
- `interact` — взаимодействие (4 кадра)

**Стиль:** Pixel art, вид сверху

**Цветовая схема:**
- Основной: #6c5ce7 (фиолетовый)
- Акцент: #a29bfe (светло-фиолетовый)
- Детали: #2d2d44 (темно-синий)

## Конвейер ассетов

### Импорт спрайтов

```typescript
// В PreloadScene
this.load.spritesheet('player', 'assets/sprites/characters/player.png', {
  frameWidth: 48,
  frameHeight: 64
})

// Создание анимации
this.anims.create({
  key: 'player-walk-down',
  frames: this.anims.generateFrameNumbers('player', { start: 0, end: 7 }),
  frameRate: 10,
  repeat: -1
})
```

### Импорт аудио

```typescript
// В PreloadScene
this.load.audio('main-theme', [
  'assets/audio/music/main_theme.mp3',
  'assets/audio/music/main_theme.ogg'
])

// Воспроизведение
this.sound.play('main-theme', { loop: true, volume: 0.5 })
```

## Оптимизация

### Спрайты
- Использовать texture atlases
- Объединять мелкие спрайты в один лист
- Сжимать PNG с потерями (TinyPNG)

### Аудио
- Сжимать MP3 до 128-192 kbps
- Использовать Web Audio API для звуков
- Предзагружать критичные звуки

### Шрифты
- Подгружать только нужные веса
- Использовать font-display: swap
- Кэшировать через service worker
