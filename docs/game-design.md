# Game Design Document (GDD)

## Концепция

**Название:** Office Quest: Путь от джуна до лида

**Жанр:** 2D Point-and-Click Quest / Adventure

**Платформа:** Web (Browser)

**Целевая аудитория:** IT-специалисты, люди, работающие в офисах

**USP:** Юмористический квест о реалиях IT-индустрии с выборами, влияющими на карьеру

## Сюжет

### Пролог

Игрок — Junior Developer, только что устроившийся в IT-компанию. Цель — за 1 год стать Team Lead, не сойти с ума и не попасть под сокращение.

### Акт 1: Новичок (Junior)

**Цель:** Доказать свою компетентность

**Конфликты:**
- Непонятные задачи
- Сложный код легаси
- Дедлайны
- Imposter syndrome

**Ключевые события:**
- Первое код-ревью (стресс +20)
- Первый баг на проде (стресс +30)
- Помощь коллеге (уважение +10)

### Акт 2: Бойец (Middle)

**Цель:** Стать надежным игроком команды

**Конфликты:**
- Менторинг джуниоров
- Выбор: скорость vs качество
- Политика компании
- Выгорание

**Ключевые события:**
- Провести митинг (уважение +15)
- Конфликт с продактом (выбор стороны)
- Дедлайн горит (стресс +40)

### Акт 3: Лидер (Senior → Lead)

**Цель:** Возглавить команду

**Конфликты:**
- Ответственность за команду
- Сокращения в компании
- Выбор между командой и карьерой
- Финальное испытание

**Концовки:**
1. **Lead** — уважение 75+, стресс <70
2. **Senior forever** — уважение 50-74
3. **Увольнение** — стресс 100
4. **Сокращение** — низкое уважение при сокращениях

## Механики

### 1. Карьерный путь

```
Junior → Middle → Senior → Lead
```

**Условия повышения:**
- Выполнено X квестов
- Уважение > порога
- Определенные квесты выполнены

### 2. Стресс (0-100)

**Источники:**
- Дедлайны (+10-30)
- Конфликты (+10-20)
- Баги на проде (+30)
- Работа в выходные (+40)

**Способы снижения:**
- Кофе (-5)
- Перерыв (-10)
- Игры с коллегами (-15)
- Выходные (-20)

**Эффекты:**
- 0-30: Норма
- 31-60: Усталость (медленнее работа)
- 61-90: Выгорание (ошибки в диалогах)
- 91-99: Критическое состояние
- 100: Game Over (увольнение по состоянию здоровья)

### 3. Уважение (0-100)

**Источники:**
- Выполненные квесты (+5-20)
- Помощь коллегам (+5-15)
- Код-ревью (+10)
- Решение проблем (+20)

**Потеря:**
- Проваленные квесты (-10)
- Конфликты (-5-15)
- Баги на проде (-20)

**Эффекты:**
- 0-30: Игнорирование
- 31-60: Нейтральное отношение
- 61-90: Уважение
- 91-100: Авторитет

### 4. Квесты

**Типы:**
- **Основные** — продвигают сюжет
- **Побочные** — дают уважение/ресурсы
- **Ежедневные** — рутина, стресс

**Примеры:**
- "Найди документацию"
- "Исправь баг за джуна"
- "Проведи код-ревью"
- "Разреши конфликт в команде"
- "Подготовь презентацию"

### 5. Диалоги

**Система выборов:**
- Каждый выбор влияет на стресс/уважение
- NPC запоминают выборы
- Разблокировка диалогов при высоком уважении

### 6. Инвентарь

**Типы предметов:**
- Квестовые (документы, ключи)
- Расходники (кофе, энергетики)
- Документация (книги, мануалы)

## Локации

### 1. Open Space
- Рабочие места
- Компьютеры
- Стендап-зона

### 2. Кухня
- Кофемашина (снижает стресс)
- Холодильник (квестовые предметы)
- Стол для обедов (диалоги)

### 3. Переговорка
- Митинги
- Презентации
- Конфликты

### 4. Кабинет директора
- Финальные квесты
- Серьезные диалоги

## Уровни сложности

- **Easy** — стресс растет медленно, уважение быстро
- **Normal** — баланс
- **Hard** — стресс быстро, уважение медленно, больше багов

## Достижения

- "Первый коммит" — завершить первый квест
- "Код-мастер" — 10 успешных код-ревью
- "Ментор" — помочь 5 джуниорам
- "Железные нервы" — пройти игру со стрессом <50
- "Все любят" — уважение 100 со всеми NPC
- "Lead" — получить повышение

## Монетизация (будущее)

- Бесплатная
- Возможно: косметические предметы
- Возможно: донаты на кофе команде :)

## Метрики успеха

- Время игры: 2-4 часа
- Реиграбельность: разные выборы → разные концовки
- Эмоциональный отклик: юмор, ностальгия, эмпатия
