# Game Design Document (GDD)

## Концепция

**Название:** Office Quest

**Жанр:** 2D Point-and-Click Quest / Карьерный симулятор с ассесментами

**Платформа:** Web (Browser)

**Целевая аудитория:** IT-специалисты и все, кто работает в офисах (разработчики, аналитики, HR, продакты, дизайнеры, QA)

**USP:** Игровая платформа развития компетенций — реальная матрица навыков, привязанная к игровой прогрессии. NPC проводят ассесменты через диалоги, мини-игры прокачивают soft skills, мультиплеер позволяет учиться вместе.

## Сюжет

### Пролог

Игрок — новичок, только что устроившийся в IT-компанию. В процессе работы он выбирает карьерную специализацию и проходит путь от Junior до Lead/Architect в выбранной области, не выгорая и не попадая под сокращение.

### Акт 1: Новичок (Junior)

**Цель:** Доказать свою компетентность, определиться с карьерным путём

**Конфликты:**
- Непонятные задачи
- Сложный код легаси
- Дедлайны
- Imposter syndrome

**Ключевые события:**
- Первое код-ревью (стресс +20)
- Первый баг на проде (стресс +30)
- Помощь коллеге (уважение +10)
- **Выбор карьерного пути** — NPC предлагают специализацию (respect >= 20)

### Акт 2: Специалист (Middle)

**Цель:** Стать надежным специалистом в выбранной области

**Конфликты:**
- Менторинг джуниоров
- Выбор: скорость vs качество
- Политика компании
- Выгорание
- Ассесменты по компетенциям

**Ключевые события:**
- Первый ассесмент от NPC-ментора
- Конфликт с продактом (выбор стороны)
- Открытие AI Lab / Game Room (новые локации)

### Акт 3: Лидер (Senior → Lead/Architect)

**Цель:** Достичь максимального грейда в выбранной специализации

**Конфликты:**
- Ответственность за команду / направление
- Сокращения в компании
- Финальный комплексный ассесмент

**Концовки (зависят от выбранного пути):**
1. **Top Grade** — достиг максимального грейда (AI Architect, Solution Architect, CPO, и т.д.)
2. **Senior forever** — хороший специалист, но не дотянул до топа
3. **Увольнение** — стресс 100 (burnout)
4. **Сокращение** — низкое уважение при сокращениях

## Механики

### 1. Карьерные пути (Plugin-система)

Каждый путь — отдельный data-модуль с матрицей компетенций, NPC-ассессорами и 4 грейдами.

| Путь | Грейды | NPC-наставник |
|------|--------|---------------|
| AI & ML | AI Junior → AI Middle → AI Senior → AI Architect | Петя Сеньор, Профессор Нейронов |
| Engineering | Junior Dev → Middle Dev → Senior Dev → Solution Architect | Тим Лид |
| Product | Junior PM → PM → Senior PM → CPO | Ольга Продакт |
| Design | Junior Designer → Designer → Senior Designer → Design Lead | Лёша Дизайнер |
| QA | Junior QA → QA → Senior QA → QA Architect | Маша QA |
| Analytics | Junior Analyst → Analyst → Senior Analyst → Head of Analytics | Игорь Аналитик |
| HR/People | HR Junior → HR BP → Senior HR BP → CHRO | Анна HR |
| Management | Team Lead → Eng Manager → Director → VP/CTO | Директор |

**Выбор пути:** через живое общение с NPC (не меню), каждый NPC "продаёт" свой путь в своём стиле.

**Условия повышения:**
- Средний балл по доменам компетенций >= порога
- Каждый домен >= минимального порога
- Стресс < 70

### 2. Стресс (0-100)

**Источники:**
- Дедлайны (+10-30)
- Конфликты (+10-20)
- Баги на проде (+30)
- Работа в выходные (+40)

**Способы снижения:**
- Кофе (-5)
- Перерыв (-10)
- Игры с коллегами (-15)
- Выходные (-20)

**Эффекты:**
- 0-30: Норма
- 31-60: Усталость (медленнее работа)
- 61-90: Выгорание (ошибки в диалогах)
- 91-99: Критическое состояние
- 100: Game Over (увольнение по состоянию здоровья)

### 3. Уважение (0-100)

**Источники:**
- Выполненные квесты (+5-20)
- Помощь коллегам (+5-15)
- Код-ревью (+10)
- Решение проблем (+20)

**Потеря:**
- Проваленные квесты (-10)
- Конфликты (-5-15)
- Баги на проде (-20)

**Эффекты:**
- 0-30: Игнорирование
- 31-60: Нейтральное отношение
- 61-90: Уважение
- 91-100: Авторитет

### 4. Система ассесментов

NPC-ассессоры проверяют знания игрока через диалоги с выбором ответа. Вопросы — рабочие ситуации, не экзаменационные тесты.

**Механика:**
- NPC описывает рабочую ситуацию → игрок выбирает из 3-4 вариантов
- Каждый вариант имеет score (0-100), NPC даёт feedback
- После ответа — объяснение правильного подхода (образовательная ценность)
- Сессия = 3-5 вопросов, адаптивная сложность
- Баллы накапливаются по доменам → определяют грейд

**Домены (пример, AI-путь):** ML Fundamentals, Data Engineering, Deep Learning, NLP/LLMs, Computer Vision, MLOps, System Design, AI Ethics

**Адаптивная сложность:** score 0-25 → difficulty 1, score 85+ → difficulty 4

### 5. Квесты

**Типы:**
- **Основные** — продвигают сюжет
- **Побочные** — дают уважение/ресурсы
- **Ассесмент** — проверка компетенций через NPC
- **Финальный** — комплексный ассесмент для top-грейда

### 6. Диалоги

**Система выборов:**
- Каждый выбор влияет на стресс/уважение
- NPC запоминают выборы
- Разблокировка диалогов при высоком уважении
- Ассесмент-вопросы генерируются динамически из матрицы компетенций
- AI-powered диалоги через OpenRouter (свободный текст)

### 7. Инвентарь

**Типы предметов:**
- Квестовые (документы, ключи)
- Расходники (кофе, энергетики)
- Документация (книги, мануалы)

### 8. Мини-игры (Game Room)

Аркадные автоматы в Game Room, каждый — отдельная Phaser Scene:

| Мини-игра | Механика | Прокачивает |
|-----------|----------|-------------|
| Code Review | Найди баги в коде за время | Внимательность, Engineering |
| Architecture Puzzle | Собери систему из компонентов (drag-and-drop) | System Design |
| Sprint Planning | Собери оптимальный спринт из задач | Product, Management |

Мини-игры снижают стресс (-5...-12) и дают бонусы к respect.

### 9. Мультиплеер

- Видим других игроков в офисе (аватары с никнеймами)
- Чат на кухне (chat bubbles), история сообщений
- Кухня — социальная зона (стресс -2 каждые 30 сек с другими игроками)
- Дуэли знаний (1v1 — один вопрос обоим, кто быстрее и правильнее)
- Командные ассесменты (2-4 игрока, каждый отвечает на свою часть)
- Leaderboard по карьерным путям

### 10. Достижения и бейджи

30+ достижений разных категорий: карьера, ассесменты, social, мини-игры, exploration, скрытые. Прогрессивные достижения (23/100). Popup при получении.

## Локации

### 1. Open Space
- Рабочие места, компьютеры, стендап-зона
- NPC: Тим Лид, другие коллеги
- Доска объявлений (leaderboard)

### 2. Кухня
- Кофемашина (снижает стресс)
- **Social hub** — расширенный чат в мультиплеере
- Бонус: stress -2 каждые 30 сек с другими игроками

### 3. Переговорка
- Митинги, презентации, конфликты
- NPC: Анна HR

### 4. Кабинет директора
- Финальные квесты, серьезные диалоги
- NPC: Директор

### 5. AI Lab (разблокируется при выборе AI-пути)
- NPC: Профессор Нейронов
- Whiteboard (открывает Skill Tree)
- Продвинутые ассесменты (system-design, ai-ethics, mlops)
- Тёмно-синий "технический" стиль

### 6. Game Room (доступна всем)
- Аркадные автоматы с мини-играми
- Code Review, Architecture Puzzle, Sprint Planning
- Неоновый стиль, снижение стресса

## Уровни сложности

- **Easy** — стресс растет медленно, уважение быстро
- **Normal** — баланс
- **Hard** — стресс быстро, уважение медленно, больше багов

## Достижения (примеры из 30+)

**Карьера:** "Первые шаги" (выбрал путь), "На вершине" (максимальный грейд)
**Ассесменты:** "Перфекционист" (score 100), "Сотня" (100 ответов), "Серия" (5 подряд > 80)
**Social:** "Привет, мир!" (первое сообщение), "Кофе-брейк" (5 мин на кухне с коллегами)
**Мини-игры:** "Arcade King" (3 звезды во всех), "Мастер Code Review"
**Скрытые:** "Дзен" (стресс = 0), "Jack of all trades" (30+ в 3 разных путях)

## Победный экран

При достижении top-грейда: radar-диаграмма компетенций по 8 осям, общий балл, время прохождения, кнопка "Экспорт отчёта" (JSON для HR).

## Монетизация (будущее)

- Бесплатная
- Возможно: косметические предметы, аватары
- Возможно: корпоративная лицензия (кастомные матрицы компетенций)

## Метрики успеха

- Время прохождения одного пути: 2-4 часа
- Реиграбельность: 8 карьерных путей, разные концовки
- Образовательная ценность: реальные компетенции, актуальные best practices
- Мультиплеер: среднее время на кухне, количество дуэлей
