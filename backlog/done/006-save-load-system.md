# Сохранение прогресса

## Описание
Реализовать сохранение/загрузку игры: позиция, инвентарь, квесты, карьерный уровень.

## Критерии приемки
1. Автосохранение при важных событиях
2. Ручное сохранение через меню
3. Загрузка при старте игры
4. Сохраняется весь прогресс
5. Несколько слотов сохранения

## Технические детали
- localStorage для веб-версии
- Создать `SaveManager` класс
- Сериализация `GameState`
- Версионирование сохранений
