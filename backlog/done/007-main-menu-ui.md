# Главное меню и UI

## Описание
Создать главное меню, паузу, настройки (звук, управление), экраны окончания игры.

## Definition of Ready (DOR)
- [x] Задача четко сформулирована
- [x] Определены критерии приемки
- [x] Оценена сложность (Story Points)
- [x] Нет блокирующих зависимостей
- [x] Определены необходимые ресурсы/ассеты

## Definition of Done (DOD)
- [x] Код написан и работает
- [x] Код соответствует стайлгайду
- [x] Функционал протестирован вручную
- [x] Нет консольных ошибок
- [x] Документация обновлена (если требуется)
- [x] Код прошел ревью (если требуется)

## Критерии приемки
1. [x] Главное меню: Новая игра, Продолжить, Настройки
2. [x] Меню паузы (Esc)
3. [x] Настройки: громкость, управление
4. [x] Game Over экран с причиной
5. [x] Victory экран при достижении лида

## Технические детали
- Создать `MenuScene`, `PauseScene`, `GameOverScene`
- UI кнопки с hover эффектами
- Анимации переходов между сценами
- Сохранение настроек

## Зависимости
- Нет

## Оценка
- Story Points: 5
- Приоритет: Medium

## Метки
- `feature`, `ui`
