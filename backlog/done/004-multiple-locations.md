# Несколько локаций офиса

## Описание
Создать 3-4 различные локации офиса: open space, кухня, переговорка, кабинет директора.

## Definition of Ready (DOR)
- [x] Задача четко сформулирована
- [x] Определены критерии приемки
- [x] Оценена сложность (Story Points)
- [x] Нет блокирующих зависимостей
- [x] Определены необходимые ресурсы/ассеты

## Definition of Done (DOD)
- [x] Код написан и работает
- [x] Код соответствует стайлгайду
- [x] Функционал протестирован вручную
- [x] Нет консольных ошибок
- [x] Документация обновлена (если требуется)
- [ ] Код прошел ревью (если требуется)

## Критерии приемки
1. Минимум 3 разные локации
2. Переходы между локациями (двери)
3. Каждая локация имеет уникальные объекты
4. NPC привязаны к локациям
5. Мини-карта локаций

## Технические детали
- Каждая локация = отдельная Scene или Room
- Переходы через дверные объекты
- Управление камерой при переходах
- Сохранение позиции игрока

## Зависимости
- Нет

- Story Points: 8
- Приоритет: Medium
