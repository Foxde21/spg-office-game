# Система стресса и уважения

## Описание
Реализовать механики стресса и уважения как ключевых параметров игрока. Влияют на геймплей и концовки.

- [ ] Код прошел ревью (если требуется)

## Критерии приемки
1. Стресс и уважение отображаются в UI
2. Действия меняют показатели (+/-)
3. Стресс 100% = game over (увольнение)
4. Уважение влияет на диалоги NPC
5. Визуальная индикация критических значений

## Технические детали
- Добавить в `GameState`:
  - `stress: number (0-100)`
  - `respect: number (0-100)`
- Источники стресса: дедлайны, баги, конфликтные диалоги
- Источники уважения: выполненные квесты, помощь коллегам
- Способы снижения стресса: кофе, перерывы, игры
