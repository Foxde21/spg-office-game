# Главное меню и UI

## Описание
Создать главное меню, паузу, настройки (звук, управление), экраны окончания игры.

## Definition of Ready (DOR)
- [x] Задача четко сформулирована
- [x] Определены критерии приемки
- [x] Оценена сложность (Story Points)
- [x] Нет блокирующих зависимостей
- [x] Определены необходимые ресурсы/ассеты

## Definition of Done (DOD)
- [ ] Код написан и работает
- [ ] Код соответствует стайлгайду
- [ ] Функционал протестирован вручную
- [ ] Нет консольных ошибок
- [ ] Документация обновлена (если требуется)
- [ ] Код прошел ревью (если требуется)

## Критерии приемки
1. Главное меню: Новая игра, Продолжить, Настройки
2. Меню паузы (Esc)
3. Настройки: громкость, управление
4. Game Over экран с причиной
5. Victory экран при достижении лида

## Технические детали
- Создать `MenuScene`, `PauseScene`, `GameOverScene`
- UI кнопки с hover эффектами
- Анимации переходов между сценами
- Сохранение настроек

## Зависимости
- Нет

## Оценка
- Story Points: 5
- Приоритет: Medium
