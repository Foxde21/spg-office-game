# Сетевая настройка для мультиплеера

## Как подключиться с другого компа в одной Wi-Fi сети

### Шаг 1: Найдите IP адрес основного компа (где запущен сервер)

На macOS (терминал):
```bash
ifconfig | grep "inet " | grep -v 127.0.0.1 | grep -v "::1"
```

Ищите адрес вроде `192.168.x.x` или `10.0.x.x`

### Шаг 2: На другом компе откройте браузер

Замените `192.168.1.100` на IP основного компа:

```
http://192.168.1.100:3000
```

### Шаг 3: Убедитесь что оба компа в одной сети

- Оба подключены к одному Wi-Fi роутеру
- Брандмауэр не блокирует порты 3000 и 3001
- На макбуке система может запросить разрешение (System Preferences → Security)

## Автоматическое определение IP

При запуске сервер выведет все доступные адреса:

```
Server running on:
  - http://localhost:3001
  - http://192.168.1.100:3001
  - http://10.0.0.50:3001
```

Используйте адрес, который соответствует вашей сети.

## Если не работает

1. Проверьте что оба компа видят друг друга:
   ```bash
   ping 192.168.1.100
   ```

2. Проверьте что сервер слушает на нужном портре:
   ```bash
   lsof -i :3000
   lsof -i :3001
   ```

3. Отключите брандмауэр временно для теста (не рекомендуется постоянно!)

## Переменные окружения

`.env.network` - шаблон для сетевой работы (0.0.0.0 = все интерфейсы)
`.env` - используется для локального dev (localhost)
